The Cannon Ball-Z

Tags:
SolidWorksAutoCAD3D-PrintingLaser CuttingManufacturing

The following project was completed as a requirement to the ME-100 course during my 1A term. The Cannon-Ball Z is an interactive toy intended for youths aged 8-12 years old. The design uses the principles of gear ratios and mechanisms along with electronic circuits that is programmed using Arduino to deliever the best entertainment. To play, the player flips switch which intiaties a motor within the toy. The electric motor rotates the internal gears that are connected in mesh to spin the shafts to which the obstacles (illustrated as red slots) are connected to. These gears are also connected to a rack and pinion gear system located at the top of the toy which allows the hoop to translate laterally. The Arduino is used to periodically reverse the rotation of the motor to allow the hoop to osicillate back and forth. Once the hoop is moving, the player tries to launch a small spherical projectile using a spring launcher and score the projectile through the hoop, which earns them points.

Sarcasm Detector

Tags:PythonMachine Learning

The Sarcasm Detector project is a natural language processing model that predicts whether a particular headline from the news is sarcastic (1) or not (0). The input data is a string of words and their expected output (sarcastic or not). Before feeding the input data into the neural network model, the input data is tokenized using the Tensorflow tokenizer class. The tokenizer class maps each word to a unique integer. More broadly, a string of words is converted into a vector where each integer represents a specific word. The input data is fed through one hidden layer with 24 units. The output of the model is a numerical value between 0 to 1. Value closer to 0 indicates the input data is not sarcastic and value closer to 1 indicates that the input data is sarcastic.
